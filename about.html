About the Project

AutoMark App is an intelligent grading assistant built using Flutter, Firebase, and Google ML Kit. It simplifies the marking and grading process of student scripts through OCR-powered scanning and instant result generation.

How the ML & OCR Works

We use Google's ML Kit Text Recognition API to scan handwritten or printed text from question papers, marking guides, and answer sheets. The recognized answers are then matched against correct responses stored in the app's Firebase database. This allows for:

Instant grading based on predefined answers
Highlighting of wrong and correct answers
Exporting of results and summary reports
Detailed analytics on student performance
Customizable grading rubrics
